# Wedding Website — Ashley & Eoin

## Project

Wedding website for Ashley & Eoin. August 20–22, 2026.
Ballintubbert Gardens & House, County Laois, Ireland.

## Full specification

Read this before writing any code:
`creative/Wedding website - Claude Code handoff.md` in my Obsidian vault

## Stack

- Plain HTML, CSS, JavaScript — no framework
- GSAP + ScrollTrigger via CDN for animations
- CSS custom properties for all design tokens

## Rules for this project

- Build structure first, animation second — never both at once
- Never use transition: all
- Never use ease-in for entrances
- Always add prefers-reduced-motion fallbacks
- Images always need explicit width and height
- Ask before making decisions that affect the whole project
